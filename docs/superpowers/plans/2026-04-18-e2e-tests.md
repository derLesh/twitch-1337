# E2E Test Harness Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add end-to-end integration tests that drive the full Rust Twitch bot against a fake IRC transport, with clock, LLM, and HTTP fakes, covering command dispatch, cooldowns, persistence, AI, aviation commands, flight tracking, and the 1337 daily handler.

**Architecture:** Split `main.rs` into `main.rs` (thin shell) + `lib.rs` (generic `run_bot<T: Transport>`). Add `Clock` trait injected surgically into wall-clock-dependent handlers. Introduce `lib::Services` bundle for test-overridable dependencies. Tests in `tests/` use a custom `FakeTransport` over `tokio::io::duplex`, a `FakeClock`, a `FakeLlm`, and `wiremock` for HTTP.

**Tech Stack:** Rust 2024 edition, `tokio`, `twitch-irc` 5.0.1, `reqwest`, `chrono`, `wiremock` (new dev-dep), `serial_test` (new dev-dep), `tempfile` (existing).

**Workflow rule:** Before every commit in this plan, run the `simplify` skill on the changed files and address its findings. Do this as a step in each task before the `git commit` step.

## Known Constraint Discovered During Planning

`twitch_irc::transport::Transport::new()` takes no arguments. Tests cannot pass a `DuplexStream` pair at construction. Workaround: global `Mutex<Option<FakeTransportInner>>` slot populated by the test before building the client; `FakeTransport::new()` drains it. Tests run serially via `serial_test::serial` to avoid slot races. Suite stays under 10s even serialized.

```rust
//! Wall-clock abstraction for time-dependent handlers.
//!
//! Production code uses `SystemClock`, which delegates to `chrono::Utc::now`
//! and `tokio::time::sleep`. Tests substitute a fake clock that the test
//! body advances manually, paired with `tokio::time::pause` so tokio-native
//! sleeps also advance deterministically.

use async_trait::async_trait;
use chrono::{DateTime, Utc};

#[async_trait]
pub trait Clock: Send + Sync {
    fn now_utc(&self) -> DateTime<Utc>;
    async fn sleep_until(&self, target: DateTime<Utc>);
}

#[derive(Debug, Default)]
pub struct SystemClock;

#[async_trait]
impl Clock for SystemClock {
    fn now_utc(&self) -> DateTime<Utc> {
        Utc::now()
    }

    async fn sleep_until(&self, target: DateTime<Utc>) {
        let delta = (target - Utc::now())
            .to_std()
            .unwrap_or(std::time::Duration::ZERO);
        tokio::time::sleep(delta).await;
    }
}
```

- [ ] **Step 2: Define `run_bot`**

```rust
use tokio::sync::{broadcast, oneshot};
use twitch_irc::login::LoginCredentials;
use twitch_irc::message::ServerMessage;
use twitch_irc::transport::Transport;

/// Run the bot until `shutdown` fires or a handler exits.
///
/// This is the entry point shared by `main.rs` and the integration test
/// harness. Production wires a real `TwitchIRCClient<SecureTCPTransport, _>`;
/// tests wire a `TwitchIRCClient<FakeTransport, StaticLoginCredentials>`.
pub async fn run_bot<T, L>(
    client: Arc<twitch_irc::TwitchIRCClient<T, L>>,
    incoming: tokio::sync::mpsc::UnboundedReceiver<ServerMessage>,
    config: crate::Configuration,
    services: Services,
    shutdown: oneshot::Receiver<()>,
) -> eyre::Result<()>
where
    T: Transport,
    L: LoginCredentials,
{
    let (broadcast_tx, _) = broadcast::channel::<ServerMessage>(100);

    let router = tokio::spawn(crate::handlers::router::run_message_router(
        incoming,
        broadcast_tx.clone(),
    ));

    let tracker_1337 = tokio::spawn(crate::handlers::tracker_1337::run_1337_handler(
        broadcast_tx.clone(),
        client.clone(),
        config.twitch.channel.clone(),
        services.clock.clone(),
        services.data_dir.clone(),
    ));

    let latency = {
        let seed = config.twitch.expected_latency;
        let latency_atomic = std::sync::Arc::new(std::sync::atomic::AtomicU32::new(seed));
        tokio::spawn(crate::handlers::latency::run_latency_handler(
            client.clone(),
            broadcast_tx.clone(),
            latency_atomic,
        ))
    };

    // Flight tracker: mpsc command channel consumed by run_flight_tracker; sender handed
    // to generic command handler so !track / !flights can send commands into it.
    let (tracker_tx, tracker_rx) = tokio::sync::mpsc::channel(32);
    let flight_tracker = if let Some(aviation) = services.aviation.clone() {
        tokio::spawn(crate::flight_tracker::run_flight_tracker(
            tracker_rx,
            client.clone(),
            config.twitch.channel.clone(),
            aviation,
            services.data_dir.clone(),
            services.clock.clone(),
        ))
    } else {
        tokio::spawn(async { /* no-op */ })
    };

    let ping_manager = crate::ping::PingManager::load(&services.data_dir).await?;
    let ping_manager = std::sync::Arc::new(tokio::sync::RwLock::new(ping_manager));
    let cooldowns = std::sync::Arc::new(tokio::sync::Mutex::new(
        std::collections::HashMap::<String, std::time::Instant>::new(),
    ));
    let leaderboard = std::sync::Arc::new(tokio::sync::RwLock::new(
        crate::handlers::tracker_1337::load_leaderboard(&services.data_dir).await,
    ));

    let cmd_cfg = crate::handlers::commands::CommandHandlerConfig {
        broadcast_tx: broadcast_tx.clone(),
        client: client.clone(),
        ai_config: config.ai.clone(),
        llm: services.llm.clone(),
        leaderboard: leaderboard.clone(),
        ping_manager: ping_manager.clone(),
        hidden_admin_ids: config.twitch.hidden_admins.clone().unwrap_or_default(),
        default_cooldown: config.pings.cooldown.unwrap_or(300),
        pings_public: config.pings.public.unwrap_or(false),
        cooldowns,
        tracker_tx,
        aviation_client: services.aviation.clone(),
        admin_channel: config.twitch.admin_channel.clone(),
        bot_username: config.twitch.username.clone(),
        channel: config.twitch.channel.clone(),
        data_dir: services.data_dir.clone(),
    };
    let commands = tokio::spawn(crate::handlers::commands::run_generic_command_handler(cmd_cfg));

    let schedules = {
        let schedule_cache = std::sync::Arc::new(tokio::sync::RwLock::new(
            crate::database::ScheduleCache::new(),
        ));
        schedule_cache.write().await.update(
            crate::handlers::schedules::load_schedules_from_config(&config),
        );
        tokio::spawn(crate::handlers::schedules::run_scheduled_message_handler(
            client.clone(),
            schedule_cache,
            services.clock.clone(),
        ))
    };

    tokio::select! {
        _ = shutdown => Ok(()),
        _ = router => Err(eyre::eyre!("router exited")),
        _ = tracker_1337 => Err(eyre::eyre!("1337 handler exited")),
        _ = latency => Err(eyre::eyre!("latency handler exited")),
        _ = flight_tracker => Err(eyre::eyre!("flight tracker exited")),
        _ = commands => Err(eyre::eyre!("command handler exited")),
        _ = schedules => Err(eyre::eyre!("schedule handler exited")),
    }
}
```

Fill in every handler spawn following the pattern. Each handler receives `broadcast_tx.clone()`, `client.clone()`, any config it needs, and `services.clock.clone()` if it was clock-injected.

```rust
//! A fake `twitch_irc::Transport` backed by `tokio::io::duplex`.
//!
//! Tests call `install(handles)` before constructing the `TwitchIRCClient`.
//! `FakeTransport::new()` drains the global slot to obtain its end of the
//! duplex stream. Because `Transport::new()` takes no arguments, this
//! ambient-state dance is unavoidable. Tests run serially
//! (`#[serial_test::serial]`) to avoid slot collisions.

use std::fmt::{self, Debug};
use std::pin::Pin;
use std::sync::Mutex;
use std::task::{Context, Poll};

use async_trait::async_trait;
use either::Either;
use futures_util::sink::Sink;
use futures_util::stream::{FusedStream, Stream};
use once_cell::sync::Lazy;
use tokio::io::{AsyncReadExt, AsyncWriteExt, DuplexStream};
use tokio::sync::mpsc;
use twitch_irc::message::{IRCMessage, IRCParseError};
use twitch_irc::transport::Transport;

static SLOT: Lazy<Mutex<Option<FakeTransportInner>>> = Lazy::new(|| Mutex::new(None));

pub struct TransportHandle {
    pub inject: mpsc::Sender<String>,
    pub capture: mpsc::Receiver<String>,
    _shutdown: tokio::sync::oneshot::Sender<()>,
}

pub(crate) struct FakeTransportInner {
    stream: DuplexStream,
    shutdown_rx: tokio::sync::oneshot::Receiver<()>,
}

pub fn install() -> TransportHandle {
    let (client_side, test_side) = tokio::io::duplex(8192);
    let (inject_tx, mut inject_rx) = mpsc::channel::<String>(64);
    let (capture_tx, capture_rx) = mpsc::channel::<String>(64);
    let (shutdown_tx, shutdown_rx) = tokio::sync::oneshot::channel();

    // Spawn driver: reads from test_side, forwards outgoing lines to capture_tx;
    // reads from inject_rx, writes incoming lines to test_side.
    tokio::spawn(async move {
        let (mut test_read, mut test_write) = tokio::io::split(test_side);
        let mut buf = Vec::with_capacity(1024);
        let read_task = async {
            loop {
                let mut chunk = [0u8; 1024];
                match test_read.read(&mut chunk).await {
                    Ok(0) => break,
                    Ok(n) => {
                        buf.extend_from_slice(&chunk[..n]);
                        while let Some(pos) = buf.iter().position(|b| *b == b'\n') {
                            let line = String::from_utf8_lossy(&buf[..=pos]).into_owned();
                            buf.drain(..=pos);
                            let line = line.trim_end_matches(['\r', '\n']).to_owned();
                            if capture_tx.send(line).await.is_err() {
                                return;
                            }
                        }
                    }
                    Err(_) => break,
                }
            }
        };
        let write_task = async {
            // handshake pre-script
            let handshake = [
                ":tmi.twitch.tv 001 bot :Welcome, GLHF!\r\n",
                ":tmi.twitch.tv 002 bot :Your host is tmi.twitch.tv\r\n",
                ":tmi.twitch.tv 003 bot :This server is rather new\r\n",
                ":tmi.twitch.tv 004 bot :-\r\n",
                ":tmi.twitch.tv 375 bot :-\r\n",
                ":tmi.twitch.tv 372 bot :You are in a maze of twisty passages.\r\n",
                ":tmi.twitch.tv 376 bot :>\r\n",
                "@emote-sets=0 :tmi.twitch.tv GLOBALUSERSTATE\r\n",
            ];
            for line in handshake {
                if test_write.write_all(line.as_bytes()).await.is_err() {
                    return;
                }
            }
            while let Some(line) = inject_rx.recv().await {
                let mut out = line;
                if !out.ends_with("\r\n") {
                    out.push_str("\r\n");
                }
                if test_write.write_all(out.as_bytes()).await.is_err() {
                    return;
                }
            }
        };
        tokio::select! {
            _ = read_task => {},
            _ = write_task => {},
        }
    });

    {
        let mut slot = SLOT.lock().unwrap();
        *slot = Some(FakeTransportInner {
            stream: client_side,
            shutdown_rx,
        });
    }

    TransportHandle {
        inject: inject_tx,
        capture: capture_rx,
        _shutdown: shutdown_tx,
    }
}

#[derive(Debug)]
pub struct FakeTransport {
    incoming: IncomingStream,
    outgoing: OutgoingSink,
}

// ... Implement `Transport` for `FakeTransport` by framing `IRCMessage` from
//     the client-side duplex half. The Incoming/Outgoing types wrap the
//     DuplexStream halves with a simple line-oriented codec (LinesCodec or
```

```rust
use std::sync::{Arc, Mutex};

use async_trait::async_trait;
use chrono::{DateTime, Duration, Utc};
use tokio::sync::oneshot;

use twitch_1337::clock::Clock;

struct Waiter {
    target: DateTime<Utc>,
    tx: oneshot::Sender<()>,
}

pub struct FakeClock {
    now: Mutex<DateTime<Utc>>,
    waiters: Mutex<Vec<Waiter>>,
}

impl FakeClock {
    pub fn new(now: DateTime<Utc>) -> Arc<Self> {
        Arc::new(Self {
            now: Mutex::new(now),
            waiters: Mutex::new(Vec::new()),
        })
    }

    pub fn set(&self, t: DateTime<Utc>) {
        *self.now.lock().unwrap() = t;
        self.wake_due(t);
    }

    pub fn advance(&self, delta: Duration) {
        let mut guard = self.now.lock().unwrap();
        *guard = *guard + delta;
        let t = *guard;
        drop(guard);
        self.wake_due(t);
    }

    fn wake_due(&self, now: DateTime<Utc>) {
        let mut waiters = self.waiters.lock().unwrap();
        let (due, pending) = waiters.drain(..).partition::<Vec<_>, _>(|w| w.target <= now);
        *waiters = pending;
        drop(waiters);
        for w in due {
            let _ = w.tx.send(());
        }
    }
}

#[async_trait]
impl Clock for FakeClock {
    fn now_utc(&self) -> DateTime<Utc> {
        *self.now.lock().unwrap()
    }

    async fn sleep_until(&self, target: DateTime<Utc>) {
        let now = *self.now.lock().unwrap();
        if target <= now {
            return;
        }
        let (tx, rx) = oneshot::channel();
        self.waiters.lock().unwrap().push(Waiter { target, tx });
        let _ = rx.await;
    }
}
```

```rust
//! Build IRCv3-tagged PRIVMSG lines for injection.

pub fn privmsg(channel: &str, user: &str, text: &str) -> String {
    privmsg_with(channel, user, text, &[])
}

pub fn privmsg_with(
    channel: &str,
    user: &str,
    text: &str,
    extra_tags: &[(&str, &str)],
) -> String {
    let mut tags = vec![
        ("badges", ""),
        ("color", ""),
        ("display-name", user),
        ("emotes", ""),
        ("id", "00000000-0000-0000-0000-000000000000"),
        ("mod", "0"),
        ("room-id", "12345"),
        ("subscriber", "0"),
        ("tmi-sent-ts", "1700000000000"),
        ("turbo", "0"),
        ("user-id", "67890"),
        ("user-type", ""),
    ];
    for (k, v) in extra_tags {
        if let Some(existing) = tags.iter_mut().find(|(name, _)| name == k) {
            existing.1 = v;
        } else {
            tags.push((k, v));
        }
    }
    let tag_str = tags
        .iter()
        .map(|(k, v)| format!("{k}={v}"))
        .collect::<Vec<_>>()
        .join(";");
    format!(
        "@{tag_str} :{user}!{user}@{user}.tmi.twitch.tv PRIVMSG #{channel} :{text}"
    )
}

pub fn privmsg_as_broadcaster(channel: &str, user: &str, text: &str) -> String {
    privmsg_with(channel, user, text, &[("badges", "broadcaster/1"), ("mod", "0")])
}

pub fn privmsg_as_mod(channel: &str, user: &str, text: &str) -> String {
    privmsg_with(channel, user, text, &[("mod", "1")])
}

pub fn parse_privmsg_text(raw: &str) -> String {
    // Input example: "PRIVMSG #chan :reply text"
    if let Some(idx) = raw.find(" :") {
        raw[idx + 2..].trim_end_matches(['\r', '\n']).to_owned()
    } else {
        raw.to_owned()
    }
}
```
